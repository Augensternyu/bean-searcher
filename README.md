# Bean Searcher

### Install 

##### Gradle

```
dependencies {
    compile 'com.ejlchina:bean-searcher:1.3.1'
}

```

##### Maven

```
<dependencies>
	<dependency>
		<groupId>com.ejlchina</groupId>
		<artifactId>bean-searcher</artifactId>
		<version>1.3.1</version>
	</dependency>
</dependencies>
```
